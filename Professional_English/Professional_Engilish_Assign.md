# MFA 专业硕士英语教学安排

## 1. 教学内容

- 1~6 单元 Text A 为重点学习内容

- 围绕 1~6 单元 Text A 进行话题推展，口头报告等口语活动

- 7~10 单元 Text A 以学生自学视频为主

- 课堂讨论作文和翻译方法

## 2. 成绩评定

- 平时：20%（课堂 10% + 网络 10%）

- 口头报告：10%

- 口试：20%

- 笔试：50%

  - 词汇：20%

  - 阅读：40%

  - 翻译：20%（英译汉 10% + 汉译英 10%）

  - 写作：20%

为确保有效学习网络课程，词汇和翻译部分 70% 来自 1~6 单元，30% 来自 7~10 单元。
